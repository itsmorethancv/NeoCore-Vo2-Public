# NeoCore Vo2 Public

NeoCore Vo2 Public is a great desktop AI tool made by Cherry Valentine. It lives on your PC and gives you a fast local HUD-style interface for chatting, running commands, using agent workflows, and interacting with your machine from a futuristic desktop command layer.

## What It Is

NeoCore is a Windows desktop HUD-style AI assistant built to feel like an advanced on-device command layer.

It is designed around the idea of a tactical HUD AI, with an AI council system that includes another AI named `Ultron` as part of the council. The experience is meant to feel like a powerful desktop intelligence system sitting on top of your PC.

NeoCore runs locally on your computer and is designed to be fast, practical, and easy to launch once set up.

## Core Capabilities

- Advanced HUD-style AI interface for local desktop use.
- Built-in AI council flow, including `Ultron`.
- Chat-based assistance with streamed responses.
- Terminal command execution from inside the app.
- Dynamic HUD widgets and live UI modules.
- Local file and coding workflows through its agent system.
- Can perform higher-trust actions such as editing or deleting files when those actions are approved.
- Can control parts of your PC workflow through its available automation and system actions.
- Includes extra utility modules like timer and stopwatch.

## Prerequisites

Before running NeoCore, make sure these are installed:

- Python
- Ollama

`initialize.bat` can help install Python and Node.js on Windows if they are missing, but Ollama should also be installed and available on the machine for the local AI workflow.

## How To Run

Clone the repository:

```bash
git clone https://github.com/itsmorethancv/NeoCore-Vo2-Public.git
```

Go into the project folder:

```bash
cd NeoCore-Vo2-Public
```

For first-time setup, run:

```bat
initialize.bat
```

After the first time, just run:

```bat
run.bat
```

## Notes

- Use `initialize.bat` the first time to install what the app needs and launch it.
- Use `run.bat` after that for normal startup.
- NeoCore is meant to run on your PC as a local desktop tool.
- The project is centered around an advanced HUD AI experience with `Ultron` in the council.
