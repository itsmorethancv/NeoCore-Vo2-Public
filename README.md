# NeoCore Vo2 Public

NeoCore Vo2 Public is a great desktop AI tool made by Cherry Valentine. It lives on your PC and gives you a fast local HUD-style interface for chatting, running commands, and using built-in utilities directly from your machine.

## What It Is

NeoCore is a Windows desktop application built to feel like a personal on-device command and assistant layer. It runs locally on your computer and is designed to be fast, practical, and easy to launch once set up.

## How To Run

Clone the repository:

```bash
git clone https://github.com/itsmorethancv/NeoCore-Vo2-Public.git
```

Go into the project folder:

```bash
cd NeoCore-Vo2-Public
```

For first-time setup, run:

```bat
initialize.bat
```

After the first time, just run:

```bat
run.bat
```

## Notes

- Use `initialize.bat` the first time to install what the app needs and launch it.
- Use `run.bat` after that for normal startup.
- NeoCore is meant to run on your PC as a local desktop tool.
